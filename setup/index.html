Setup

Setting up your Air Quality Sensor

Follow these steps to get started with your air quality sensor:

1. Connect the sensor to the compute disk.

2. Connect the usb cable to power the board

Once you've completed these steps, your air quality sensor should be up and running.

Saving data on an SD card

You can collect data over time using an SD card, but before you insert an SD card in your air quality sensor, you need to setup the file structure and add the config files.

Config file generation

This section provides instructions for generating a configuration file for your air quality sensor using the information you provide. Please note that this data is kept local and is never transmitted to the internet.

Use the form below to generate a configuration file based on the information you input. The config file, called "wifi_credentials.txt" must be placed in a folder named "config" within the root directory of the SD card. If the "config" folder does not already exist, create one before copying the "wifi_credentials.txt" file into it.






Template download

The air quality sensor serves a web interface directly from the SD card. This is architected this way to allow you to easily customize the web interface. You can download the default template here and add it to the root of your SD card.
